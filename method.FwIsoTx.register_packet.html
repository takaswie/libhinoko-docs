Description [src]

Register packet data with header and payload for IT context. The content of given header and payload is appended into data field of isochronous packet to be sent. The caller can schedule hardware interrupt to generate interrupt event. In detail, please refer to documentation about HinokoFwIsoTx::interrupted.

Available since:0.7.
