Description [src]

Register chunk of buffer to process packet for future isochronous cycle. The caller can schedule hardware interrupt to generate interrupt event. In detail, please refer to documentation about HinokoFwIsoRxSingle::interrupted signal.

Available since:0.7.

  The return location for a GError*, or NULL.
