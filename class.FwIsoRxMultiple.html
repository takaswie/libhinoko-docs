HinokoFwIsoRxMultiple

Description [src] 

class Hinoko.FwIsoRxMultiple : GObject.Object {
  parent_instance: GObject
}

An object to receive isochronous packet for several channels. A HinokoFwIsoRxMultiple receives isochronous packets for several channels by IR context for buffer-fill mode in 1394 OHCI.

Instance methods

hinoko_fw_iso_rx_multiple_allocate

Allocate an IR context to 1394 OHCI controller for buffer-fill mode. A local node of the node corresponding to the given path is used as the controller, thus any path is accepted as long as process has enough permission for the path.

hinoko_fw_iso_rx_multiple_get_payload

Retrieve data for packet indicated by the index parameter. The data has isochronous packet header in its first quadlet, timestamp in its last quadlet. The rest is data of isochronous packet.

Available since: 0.7.

hinoko_fw_iso_rx_multiple_map_buffer

Map an intermediate buffer to share payload of IR context with 1394 OHCI controller.

Start IR context.
