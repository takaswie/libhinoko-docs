Description

A HinokoFwIsoCtx is an abstract object to maintain isochronous context by UAPI of Linux FireWire subsystem. All of operations utilize ioctl(2) with subsystem specific request commands. This object is designed for internal use, therefore a few method and properties are available for applications.

hinoko_fw_iso_ctx_get_cycle_timer (HinokoFwIsoCtx *self,
                                   guint16 *cycle_timer,
                                   GTimeVal *tv,
                                   GError **exception);

Retrieve the value of cycle timer register. When tv argument is given, it's filled for the nearest system time with CLOCK_MONOTONIC_RAW flag. This method call is available once any isochronous context is created.
