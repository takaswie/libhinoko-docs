Description

A HinokoFwIsoCtx is an abstract object to maintain isochronous context by UAPI of Linux FireWire subsystem. All of operations utilize ioctl(2) with subsystem specific request commands. This object is designed for internal use, therefore a few method and properties are available for applications.
