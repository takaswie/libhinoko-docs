Description [src] 

class Hinoko.FwIsoRxSingle : GObject.Object {
  parent_instance: GObject
}

An object to receive isochronous packet for single channel. A HinokoFwIsoRxSingle receives isochronous packets for single channel by IR context for packet-per-buffer mode in 1394 OHCI. The content of packet is split to two parts; context header and context payload in a manner of Linux FireWire subsystem.

Allocate an IR context to 1394 OHCI controller for packet-per-buffer mode. A local node of the node corresponding to the given path is used as the controller, thus any path is accepted as long as process has enough permission for the path.
