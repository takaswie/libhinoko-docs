Hinoko

API Version: 0.0

Library Version: 0.7.1

The library produces some objects derived from GObject as an abstraction to operate system resources in Linux FireWire subsystem.

The library operates 1394 OHCI controller for isochronous communication on IEEE 1394 bus. All operations are associated with GSource retrieved from each object, therefore applications should be programmed with GMainContext.

The library supports GObject Introspection, thus this library is available with GObject Introspection bindings of each language such as Python, Ruby and so on. When using this mechanism, please refer to documents in each language implementation as well.
